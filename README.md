# Heal Together Frontend

Frontend for the Heal Together healthcare application built using React + Vite.

## Tech Stack

- React
- Vite
- CSS
- Fetch API

## Features

- Patient health form
- Symptom selection
- Remedy suggestions
- Doctor portal UI
- Responsive interface

# Local Development

## 1. Clone Repository

git clone https://github.com/Shreya-ravi/healtogether-frontned.git

## 2. Open Project (VS)

cd healtogether-frontned

## 3. Install Dependencies

npm install

## 4. Run Frontend

npm run dev

Frontend runs on:

http://localhost:5173


# Backend Connection

Frontend connects to Render backend API:

https://healtogether-backend.onrender.com/api

Example:

const API_BASE = 'https://healtogether-backend.onrender.com/api

# Vercel Deployment

## Deploy Steps

1. Push frontend code to GitHub
2. Open Vercel
3. Import GitHub repository
4. Framework preset: Vite
5. Click Deploy


# Build Command

npm run build
